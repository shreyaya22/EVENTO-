Project: Event Management System by group:nimbus2000

Installation guide:

Download the source code and extract the zip file.
Download or set up any local web server that runs PHP script.
Open the web-server database and create a new database name it event_db.
Import the SQL file located in the database folder of the source code.
Copy and paste the source code to the location where your local web server accessing your local projects. Example for XAMPP('C:\xampp\htdocs')
Open a web browser and browse the project. E.g [http://localhost/event-management-system] for the visitor side and [http://localhost/event-management-system/admin] for the admin side.
